Introduction

Linux is a family of open-source Unix-like operating systems based on the Linux kernel.

It is one of the most prominent examples of free and open-source software collaboration.

Some common Linux distributions include Ubuntu, Fedora, CentOS, and Kali.

This documentation covers basic commands that are essential for using Linux.

Basic Commands

Some basic Linux commands include:

  • pwd: Print Working Directory
  • ls: List directory contents
  • cd: Change Directory
  • echo: Display a line of text
  • man: Manual pages

The pwd command shows the current directory you are in.

The ls command lists the files and directories in the current directory.

File Operations

File operations include creating, deleting, and editing files.

  • touch: Create an empty file
  • rm: Remove files or directories
  • cp: Copy files or directories
  • mv: Move or rename files or directories
  • cat: Concatenate and display file content

The touch command is used to create an empty file.

The rm command is used to delete files or directories.

Directory Operations

Directory operations include creating and removing directories.

  • mkdir: Create a new directory
  • rmdir: Remove an empty directory
  • cd: Change directory
  • ls: List directory contents
  • pwd: Print Working Directory

The mkdir command is used to create a new directory.

The rmdir command is used to remove an empty directory.

Permissions

Managing file and directory permissions is crucial in Linux.

  • chmod: Change file mode (permissions)
  • chown: Change file owner and group
  • chgrp: Change group ownership

The chmod command changes the file's permissions.

The chown command changes the file's owner and group.
